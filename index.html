Analysis of Joe Rogan's Podcast on Youtube
By Elaina Tiller
Joe Rogans exclusive Spotify deal took his podcasts off Youtube.
With nearly 11 year existence of The Joe Rogan Experience,
I used the Youtube API I analyzed more than 2500 videos before they got taken down

MySQl

I used MySQL to create a database with over 2500 videos with information on likes, dislikes, guests and release data of each video

Matplotlib

Created multitude of visualizations with different types of graphs and data calculations

Python

Challenged my Python skills with regex to extract specific guest names and dates
